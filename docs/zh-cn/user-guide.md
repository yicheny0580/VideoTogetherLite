# 用户指南

VideoTogether Lite 会在邀请码房间中同步你主动选择并共享的视频播放状态。

## 安装

本地测试：

1. 运行 `just setup` 和 `just build-extension`。
2. 打开 `chrome://extensions`。
3. 启用开发者模式。
4. 加载已解压的 `apps/extension/dist`。

Alpha 或 beta 测试时，请使用维护者提供的私有 Chrome Web Store 测试链接安装。

## 创建房间

1. 打开包含视频的网页。
2. 打开 VideoTogether Lite 扩展 popup。
3. 输入昵称。
4. 点击 `Create room`。
5. 复制邀请码并发送给其他观看者。

邀请码同时包含房间号和邀请密钥，请像房间密码一样保管。

## 加入房间

1. 打开包含视频的网页。
2. 打开扩展 popup。
3. 输入昵称。
4. 粘贴邀请码。
5. 点击 `Join room`。

同一个浏览器配置只能加入一个房间。创建或加入新房间会离开之前的房间。

## 选择并共享视频

1. 在包含视频的页面打开浮动面板或 popup。
2. 点击 `Pick video`。
3. 点击要共享的视频元素。
4. 在面板中开启共享。

只有被选中的视频会被共享，页面浏览行为和其他视频不会发送到房间。

## 跟随共享视频

1. 加入同一个房间。
2. 打开包含对应视频的页面。
3. 在 `Shared videos` 中选择共享者。
4. 点击 `Follow`。

部分网站会受到浏览器自动播放限制，需要先手动点击一次播放。

## 退出房间

在 popup 或面板中点击 `Exit room`。最后一个参与者离开或全部参与者超时后，后端会删除房间。

## 故障排查

- `No videos found on this page`: 等视频播放器显示后刷新页面，再重新打开 popup。
- `Need to play manually`: 在网页播放器中手动点击一次播放。
- 邀请码无效: 请让房主重新复制邀请码。
- 房间消失: 后端只在内存中保留房间，超过配置的房间 TTL 后会过期。
- popup 显示页面不可用: 切换到普通视频网页并刷新。
